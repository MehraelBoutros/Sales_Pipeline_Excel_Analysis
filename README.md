## Project Overview
This project analyzes CRM sales pipeline data to evaluate sales performance, deal conversion rates, and sales cycle efficiency. The goal is to identify key drivers of successful deals and provide recommendations to improve win rates and sales team productivity.

## Dataset Description
The dataset contains information about sales opportunities, sales agents, regions, deal stages, and closing outcomes.

Key fields include:

- Opportunity ID
- Sales Agent
- Product
- Account
- Deal Stage
- Close Date
- Close Value
- Regional Office
- Sales Cycle Time

## Tools Used
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- KPI Analysis
- Data Visualization
- Excel Dashboard

## Methodology
The analysis process included:

1. Cleaning and organizing sales pipeline data
2. Calculating conversion metrics and win rates
3. Analyzing regional sales performance
4. Measuring average sales cycle duration
5. Evaluating performance differences between sales agents
6. Identifying operational improvements for the sales process

## Key Insights

- The overall **win rate is 48.16%**, meaning nearly half of all opportunities convert to closed deals.
- The **West Region** achieved the highest win rate at **63.41%**.
- The **average successful sales cycle is 52 days**.
- Deals that extend beyond **60 days have a significantly higher probability of being lost**.
- There is a noticeable **performance gap among sales agents**, with top agents closing **8.24% of deals compared to 1.18% for the lowest performers**.

## Business Recommendations

- Implement a **time alert system** to flag deals approaching the 52-day benchmark.
- Develop targeted strategies to improve performance in lower-performing regions.
- Introduce **mentorship programs** pairing high-performing agents with lower performers.
- Create structured follow-up processes for deals exceeding 45–50 days.

## Conclusion
The analysis highlights that sales cycle management, regional strategy optimization, and agent performance alignment are key drivers for improving overall sales performance. By addressing these areas, organizations can significantly increase win rates and pipeline efficiency.
